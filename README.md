# NBA-Stats-Zone-API
NBA Stats Zone is a RESTful backend service built with Spring Boot that manages NBA player statistics.


## Tech Stack
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- SQL / H2
- Maven

## Features
- Create, read, update, and delete NBA player records
- Query players by name
- RESTful endpoints with proper HTTP methods
- Service-layer business logic with repository-based persistence

## Architecture
- **Controller** – Handles HTTP requests and responses  
- **Service** – Encapsulates business logic  
- **Repository** – Abstracts database access via JPA  
- **Entity** – Represents player data model
